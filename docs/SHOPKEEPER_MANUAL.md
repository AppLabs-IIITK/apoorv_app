# Apoorv App User Manual (Shopkeepers and Shopkeeper Mode)

This guide is for users who have been enabled as shopkeepers by Admin/Subadmin and who use `Shop Coins` (shop points) to reward users.

Important note:
- There is no seperate app or login for shopkeepers.
- Shopkeepers use the same 4-tab UI as everyone else. The difference is which buttons and modes you can access.

## Key Concepts

### Points vs Shop Coins
- `Points` (Apoorv Coins): the normal coins every user has.
- `Shop Coins`: a separate balance only for shopkeepers.
  - When you reward someone in Shopkeeper Mode, your `Shop Coins` decrease.
  - The receiver’s `Points` increase.

### What Shop Coins Are For
Shop Coins are intended for shop rewards (example: rewarding a user after a purchase or approved action).

### Shop Reward Conditions (Rules)
When using Shopkeeper Mode:
- You must have enough `Shop Coins` for the reward amount.
- You cannot reward yourself.
- Reward amount is limited to `150` points per user.
- You can reward the same user using Shop Coins only once.

### Inter-College Transfer Rules (Normal Points)
Normal point transfers still follow the regular rule:
- College users cannot send points to outside-college users.
- Outside-college users can send points to college users (if they have enough points).

## How To Start Using Shopkeeper Mode

Shopkeeper Mode is a switch inside the main app UI.

1. Open `Profile` tab.
2. Find the `Shopkeeper mode` toggle.
3. Turn it on to use `Shop Coins` for payments.
4. Turn it off anytime to go back to normal `Points`.

When Shopkeeper Mode is ON:
- Your balance label will show `Shop Coins` instead of normal `Points`.
- The payment screen will show a small hint like `Using shopkeeper points`.

Note: Ensure shopkeeper mode is turned on to use shop points or you will end up sending your points

## Screen-By-Screen (Same 4 Tabs)

## Tab 1: Home (Feed)
- Same as regular users: announcements and updates.

## Tab 2: Maps
- Same as regular users: view locations and events, filter by day, open all events, share events.

## Tab 3: Points

This screen works like the normal Points screen, with one key difference:
- If Shopkeeper Mode is enabled, the balance shown is `Shop Coins` and payments use Shop Coins.

Ways to start a reward/payment:
- Search a user (`Search Someone`) and tap them
- Scan their QR (`SCAN QR CODE`)
- Tap them from the leaderboard
- Tap them via a transaction’s details view

## Tab 4: Profile

### Shopkeeper Mode Toggle
- Use `Shopkeeper mode` toggle to switch between normal points and shop coins.

### Your QR Code
- Your QR is shown at the bottom.
- Users can scan it to pay you (normal transfers), if allowed by the transfer rules.

### Inspect (Self)
- Tap your profile picture to open your own Inspect screen.
- Use filters to review transfers and shop rewards.

## Payment Screen (Rewarding a User)

How to reward:
1. Open the payment screen for a user (search/QR/leaderboard/transaction details).
2. Enter the amount.
3. Tap `Continue`.

Inspect option:
- In the top-right corner, tap `Inspect` to open the receiver’s transaction log.

What Inspect helps with:
- Transparency and fraud checks: review the user’s transfer activity and totals to spot unusual patterns.
- Dispute handling: verify whether a reward/transfer happened and when.
